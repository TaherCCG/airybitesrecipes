Admin Panel

Welcome to the Admin Panel. As an administrator, you have access to powerful tools to manage the site efficiently. Please use these tools with care and responsibility, as they impact user experience and site functionality directly.

Your actions can influence site integrity and user trust, so exercise discretion when making changes. Thank you for your diligence in maintaining a safe and enjoyable experience for all users.

Here's what you can do:

Manage Recipes

Oversee and control all recipe-related content on the site. You can view a list of all available recipes, delete unwanted recipes, and access detailed information for each recipe.

  • View all recipes: See a list of all recipes available on the site.
  • Delete a recipe: Remove a recipe permanently from the database.
  • View recipe details: Access detailed information about any recipe.

Effective recipe management ensures that content remains fresh and relevant, contributing to a positive user experience. Carefully review recipes before making changes.

Manage Categories

Manage recipe categories to organise content effectively. You can add new categories, edit existing ones, delete categories, and view all categories currently in use.

  • Add a new category: Create a new category for recipes.
  • Edit a category: Modify the name or details of an existing category.
  • Delete a category: Remove a category along with an option to remove all recipes assigned to the ctegory.
  • View all categories: See a list of all categories currently available.

Categories play a vital role in organising recipes effectively. Ensure changes align with the site's content structure and user expectations.

Manage Users

Administer user accounts and roles to maintain site security and functionality. You can view all registered users, update their roles (e.g. from user to admin), and delete user accounts as needed.

  • View all users: Access a list of all registered users on the site.
  • Edit user roles: Update user roles to grant or restrict access to specific site features.
  • Delete a user: Permanently remove a user account from the system, option to remove all recipes created by the user.

Managing user accounts and roles is crucial for maintaining site security and ensuring a positive user experience. Handle user data with care and respect user privacy at all times.
